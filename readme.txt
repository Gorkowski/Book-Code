The files here include examples, exercises and solutions from each chapter in the “Aerosol Modelling: from theory to code” book by Topping, Bane et al. All codes discussed in a chapter, along with those required to undertake each exercise and to implement the suggested solution are given in a directory relating to that chapter.

The latest versions (and further examples and information) can always be found at 
https://github.com/aerosol-modelling/Book-Code
